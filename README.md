# Heart Disease Dashboard (Tableau)

## 📊 Project Overview

This project presents an interactive data visualization dashboard created using **Tableau** to analyze patterns in heart disease data.
The dashboard helps in understanding how different factors such as age, sleep time, and physical health relate to heart disease.

## 🛠 Tools & Technologies

* Tableau
* CSV Dataset
* Data Visualization Techniques

## 📂 Dataset

The dataset used in this project contains health-related information such as:

* Age
* Physical Health
* Sleep Time
* Skin Cancer
* Stroke
* Heart Disease Status

Dataset file: `dataset_raw.csv`

## 📈 Dashboard Features

The dashboard includes multiple visualizations to explore the data:

* Bar Chart for Stroke vs Heart Disease
* Stacked Bar Chart for Physical Health Analysis
* Sleep Time comparison
* Interactive filters for better exploration

## 🎯 Objectives

The main goals of this dashboard are:

* Understand factors influencing heart disease
* Identify patterns in health-related data
* Demonstrate data visualization skills using Tableau
